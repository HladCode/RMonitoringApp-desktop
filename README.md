# Refrigerator monitoring application for desktop
It is an application for monitoring refrigerator units data from RMonitoring-server created using C++ and Qt framework.<br>
Project is sponsored by Hlad Service.<br><br>
(Currently under development) 
<br><br>
## Features
Application provides structured data visualization in a special datetime frame or in real time. Also it has administrator tools, such as add/change/delete/etc user or unit, and authorization to access data from a specific server that can be changed in settings.
